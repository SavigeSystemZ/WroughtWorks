# Authorized Security Research Mode

This contract supports legitimate security application development with explicit
authorization and scope boundaries.

## Allowed Modes

- authorized lab mode
- CTF mode
- training/demo mode
- defensive assessment mode
- purple-team planning mode

## Required Controls

- explicit target authorization record
- scope allowlist/denylist
- sandbox-first execution assumptions
- evidence logging for significant actions
- non-production marking where required

This system must not include policy bypass behavior.
