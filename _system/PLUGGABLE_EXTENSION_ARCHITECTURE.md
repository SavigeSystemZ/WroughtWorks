# Pluggable Extension Architecture

AIAST supports optional extension modules with explicit manifests.

## Plugin Types

- agent-adapter
- archetype
- validator
- doctor
- prompt-pack
- skill
- command
- mcp-integration
- runtime-scaffold
- packaging-target
- security-profile

## Manifest Minimum

- plugin id
- plugin type
- version
- dependencies
- conflicts
- validators
- installability flags
