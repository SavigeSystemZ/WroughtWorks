# Golden Examples

This directory contains neutralized example guidance for AIAST.

## What is here

- `PATTERN_INDEX.md` — entrypoint for the example pack
- `golden-example-manifest.json` — machine-readable map of the pack
- `patterns/` — donor-derived patterns for continuity, governance, MCP, validation, and platform shaping
- `working-files/` — exemplar working files that show the expected quality bar

## What is not here

- donor runtime code
- donor secrets, tokens, ports, or provider configs
- donor product truth that would contaminate a new repo

Use `_system/GOLDEN_EXAMPLES_POLICY.md` before applying anything from this directory.
