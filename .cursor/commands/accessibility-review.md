# Accessibility Review

Run the accessibility review playbook.

## Authority

- `AGENTS.md`
- `_system/ACCESSIBILITY_STANDARDS.md`
- `_system/review-playbooks/ACCESSIBILITY_REVIEW_PLAYBOOK.md`
- `_system/PROJECT_PROFILE.md`

## Process

1. Load the accessibility standards and review playbook.
2. Identify the changed or in-scope UI components.
3. Check semantic HTML, keyboard access, ARIA usage, contrast, focus management, and form accessibility.
4. Report findings using the playbook output format.
5. Fix must-fix issues or record them in `FIXME.md`.
