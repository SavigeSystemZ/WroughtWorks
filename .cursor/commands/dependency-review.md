# Dependency Review

Run the dependency review playbook.

## Authority

- `AGENTS.md`
- `_system/DEPENDENCY_GOVERNANCE.md`
- `_system/review-playbooks/DEPENDENCY_REVIEW_PLAYBOOK.md`
- `_system/PROJECT_PROFILE.md`

## Process

1. Load the dependency governance policy and review playbook.
2. Identify new, updated, or removed dependencies.
3. Check necessity, license, security, size impact, and maintenance health.
4. Report findings using the playbook output format.
5. Flag must-fix issues or record them in `FIXME.md`.
