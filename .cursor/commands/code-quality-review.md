# Code Quality Review

Run the code quality review playbook.

## Authority

- `AGENTS.md`
- `_system/CODING_STANDARDS.md`
- `_system/review-playbooks/CODE_QUALITY_REVIEW_PLAYBOOK.md`
- `_system/PROJECT_RULES.md`

## Process

1. Load the coding standards and code quality review playbook.
2. Review changed files for naming, function design, error handling, resource management, type safety, and testing.
3. Report findings using the playbook output format.
4. Fix must-fix issues or record them in `FIXME.md`.
