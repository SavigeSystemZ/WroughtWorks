# Checkpoint Handoff

## Steps

1. Update `TODO.md`.
2. Update `FIXME.md` for newly discovered issues.
3. Update `WHERE_LEFT_OFF.md` with work done, commands run, blockers, and next best step.
4. Update `CHANGELOG.md` for user-visible or architectural changes.
5. Update `_system/context/CURRENT_STATUS.md` or `_system/context/DECISIONS.md` when needed.
6. Run validation.
7. Summarize the handoff packet clearly.

Reference `_system/CHECKPOINT_PROTOCOL.md`.
